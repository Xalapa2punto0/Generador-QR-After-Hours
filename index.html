Generador de código QR

Introduce un nombre (o cualquier texto) y genera un código QR. Descárgalo como imagen o compártelo.

Puedes escribir nombres cortos o frases. El QR encodifica exactamente el texto que pongas.

(QR se mostrará aquí)
Tamaño por defecto: 220×220 px. Ajusta el texto para cambiar el contenido.

Archivo listo para subir a GitHub Pages: guarda este archivo como index.html y súbelo al repositorio.
Si quieres que el QR apunte a una URL (por ejemplo: https://tusitio.com/?nombre=...), dímelo y lo adapto.
